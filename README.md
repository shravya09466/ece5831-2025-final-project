## Project : Voice Based Speaker Recognition Using MFCC and LSTM

## Overview
To design and implement a robust speaker identification system that can accurately recognize a speaker’s identity from raw speech audio
using machine-learning–based feature extraction and deep learning, even under realistic recording conditions.
